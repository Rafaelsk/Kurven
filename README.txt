Bienvenidos al proyecto Kurven

Este software de licencia GPL, es un simulador para el diseño y puesta a punto de un sistema de control
 de algun proceso físico con el cual se cuente alguna data disponible de una prueba escalon.

El proyecto es una carpeta de Netbeans, desde Netbeans se podra tener acceso a todo el código fuente de la interfaz gráfica en Java
Ademas para los cálculos fuertes existen módulos programados en ADA
